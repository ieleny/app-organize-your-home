# organize-your-home
Application to help organize what do you need buy for your home

# Tecnologies
- node:18-alpine
- Ant Design: https://ant.design/components/overview/
- React
- Typescript
- Jotai

# Execute docker build

Inside the docker folder execute command:
```
	docker compose up
```
